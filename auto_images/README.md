# Auto Images for Adobe Illustrator

一个用于 Adobe Illustrator 的自动化图片导入和排列脚本。

## 功能特点

- 批量导入多个图片文件
- 支持多种图片格式（JPG、PNG、TIF/TIFF、PSD）
- 自动排列图片，支持三种布局方式：
  - 网格布局（可自定义行数和列数）
  - 单行布局
  - 单列布局
- 可自定义图片间距和起始位置
- 支持多种排序方式：
  - 按文件名排序
  - 按修改时间排序
  - 按文件类型排序
- 支持横向和竖向排序方向（网格布局）
- 自动居中对齐功能

## 使用要求

- Adobe Illustrator CS6 或更高版本
- 支持的图片格式：
  - JPEG/JPG
  - PNG
  - TIF/TIFF
  - PSD

## 安装方法

1. 下载 `auto_images_v1.1.0_windows.jsx` 文件
2. 将文件复制到 Adobe Illustrator 的脚本文件夹：
   - Windows: `C:\Program Files\Adobe\Adobe Illustrator [版本]\Presets\[语言]\Scripts`
   - macOS: `/Applications/Adobe Illustrator [版本]/Presets/[语言]/Scripts`

## 使用方法

1. 在 Adobe Illustrator 中打开或创建一个文档
2. 选择 文件 > 脚本 > auto_images_v1.1.0_windows
3. 在弹出的对话框中：
   - 选择包含图片的文件夹
   - 选择需要导入的图片格式
   - 设置布局参数（布局类型、行数、列数等）
   - 设置间距和起始位置
   - 选择排序方式和方向
4. 点击"确定"开始导入和排列图片

## 参数说明

### 布局设置
- **布局类型**：选择网格、单行或单列布局
- **行数**：网格布局中的行数（仅网格布局可用）
- **列数**：网格布局中的列数（仅网格布局可用）
- **水平间距**：图片之间的水平间距（单位：像素）
- **垂直间距**：图片之间的垂直间距（单位：像素）
- **X坐标**：起始位置的X坐标
- **Y坐标**：起始位置的Y坐标

### 排序设置
- **排序方式**：
  - 名称排序：按文件名自然排序
  - 修改时间排序：按文件的最后修改时间排序
  - 文件类型排序：按文件扩展名排序
- **排序方向**：
  - 横向排序：从左到右，从上到下
  - 竖向排序：从上到下，从左到右（仅网格布局可用）

## 注意事项

1. 使用前请确保已打开或创建了 Illustrator 文档
2. 图片将保持原始尺寸导入
3. 导入的图片默认不嵌入文档中，保持外部链接状态
4. 建议在使用脚本前先备份重要文档
